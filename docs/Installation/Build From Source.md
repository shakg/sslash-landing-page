## Clone Repository
1. Navigate to [https://github.com/shakg/sslash]("https://github.com/shakg/sslash")
2. Clone repository
3. `cd sslash`
4. `npm install`
5. `npm run deploy`
6. You can see the executable in `release` folder. 
 6.1 You will see .exe if you are in windows and .AppImage if you are in linux
